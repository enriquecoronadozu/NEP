How to use Choregraphe for offline programming (Windows)

 

Choregraphe is the official Simulator for Pepper and NAO robots. This tutorial will show how to use Choregraphe for prototyping of application for Pepper and NAO when is not possible to have the robots physically.

Download choregraphe mdi-download
   

After installing Choregraphe search the naoqi-bin.exe file inside the bin folder in the installation path of Choregraphe (example C:\Program Files (x86)\Aldebaran\Choregraphe Suite 2.4\bin) and create a shortcut in the Desktop.

          

We can use this shortcut to run naoqi-bin.exe, which will be used to simulate the real robot.

Open Choregraphe and connect to the port "9559" and the IP "127.0.0.1", such as is shown in the next image.

   
  
           

Then, you can use NAO SDK (naoqi) or nepsy (using nao_aldebaran methods) to programm Pepper or NAO.

    When using the simulator you can not be able to access to the sensors of the robot. However, you can use this approach for testing robot motions and text to speech.
