Unzip this file which contains jeromq-0.5.2.jar, gson-2.8.4.jar and nep.jar java binaries. Copy and paste java binaries We need to save the java binaries in a location where matlab can always find them. In windows we recommend to create a new folder in C:\nep-matlab
 
Copy and paste the java binaries inside NEP-matlab.zip to C:\nep-matlab
 
Continue Search/create javaclasspath.txt Run the command prefdir in MATLAB. Navigate to the folder shown in the MATLAB console and check if the file javaclasspath.txt exist. If the javaclasspath.txt file is not in the folder, then create it as a new text file
 
Continue Edit javaclasspath.txt Open the file javaclasspath.txt with a text editor. We needed to write the full path of the jar files jeromq-0.5.2.jar, gson-2.8.4.jar and nep.jar
 
If files are in C:\nep-matlab we need to add the next lines:
 
{{path_matlab}}
Save and close javaclasspath.txt file
