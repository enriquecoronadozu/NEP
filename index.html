Unzip this file which contain jeromq-0.4.3.jar, gson-2.8.4.jar and nep.jar java binaries. Copy and paste java binaries We need to save the java binaries in some locations where matlab can always find it. In windows we recommend to create a new folder in C:\nep-matlab
 
Copy and paste the java binaries inside NEP-matlab.zip to C:\nep-matlab
 
Continue Search/create javaclasspath.txt Run the command prefdir in MATLAB. Navigate to the folder shown in the MATLAB console and check is a file javaclasspath.txt exist. If the javaclasspath.txt file is not in the folder, then create it as a new text file
