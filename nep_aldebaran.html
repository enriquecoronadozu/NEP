What is nep_aldebaran?

 

nep_aldebaran is an high-level implementation of the NAO/Pepper SDK

Requirements:
 

To use nep_aldebaran classes it is required to install the NAO/Pepper SDK. For Windows, you can use the installer of NAO SDK 2.4.3.2 from the next link:

 
   NAO SDK 2.4.3.2
 

For other platforms or versions of NAO/Pepper SDK, please visit the SoftBank Robotics official web page

 
NAO/Pepper SDK can only be executed in Python 2 see NAO SDK documentation for each platform
