Robotics for everyone

 

NEP + is a open-source, user-friendly and cross-platform toolkit for building interactive experiences with robots

 
 
 
 
  
 
 

Connecting code and things

 

With NEP + developers can create links between robotics building blocks.

 
 
 
 
 

Write once, run on any computer

 

Most tools of NEP + are compatible with Windows, OSX and Linux

           

NEP + was designed to be simple, easy-to-install, easy-to-use and easy-to-learn. NEP + is mainly build on top of high-performance and usable sockets libraries, popular development frameworks and modern programming languagues including:
